SQL Queries 1

Show the information of movies (Title, Region, Language, Year, Rating, Votes) which have average rating greater than 7 and number of votes greater than 1000.

Statement:
SELECT tb.primaryTitle, ta.region, ta.language, tb.startYear, tr.averageRating, tr.numVotes
FROM Title_basics tb NATURAL JOIN Title_akas ta NATURAL JOIN Title_ratings tr
WHERE averageRating>7 AND numVotes>1000
LIMIT 10;

SQL Queries 2

b. Show the names of persons who have directed more than 10 movies.

Statement:
SELECT nb.primaryName
FROM Name_basic nb INNER JOIN Title_crew tc ON nb.tconst=tc.directors
WHERE nb.knownForTitle LIKE '%,%,%,%,%,%,%,%,%,%'
LIMIT 10;

SQL Queries 3

Show the top 10 TV series (more than 1 season) which most average ratings.

Statement:
SELECT DISTINCT tb.primaryTitle
FROM Episode e
INNER JOIN Title_basic tb ON e.parentTconst=tb.tconst
WHERE e.seasonNUMBER>1
LIMIT 10;

SQL Queries 4

You need to design at least FOUR more queries which satisfy use aggregate functions, “group by” and “order by”.

Statement:
SELECT primaryName
FROM Name_basic
WHERE deathYEar='NULL'
GROUP BY primaryName
LIMIT 10;

SQL Queries 5

You need to design at least FOUR more queries which satisfy use set operations.

Statement:
SELECT primaryName
FROM Name_basic
UNION
SELECT title
FROM Title_akas tb.primaryTitle
GROUP BY title
LIMIT 10;
